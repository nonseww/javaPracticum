1. "Добавь <Название города | Название города | ...>" - добавляет один или несколько городов с указанным названием
    в список ранее добавленных городов. Города требуется вписывать через пробел!
2. "Выведи информацию" - выводит информацию обо всех городах в списке ранее добавленных.
3. "Выведи информацию по <Название города>" - выводит информацию только для заданного города. Берёт
    уже существующую информацию из списка ранее добавленных городов.
4. "Выведи города с <температура>" - выводит информацию про города с указанной температурой.
5. "Удали город <Название города | Название города | ...>" - удаляет заданные города из списка ранее добавленных
    городов. Города требуется вписывать через пробел!
6. "Очисти список" - очищает список ранее добавленных городов.
7. "Команды" - выводит список доступных команд.
