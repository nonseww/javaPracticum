1. "Добавить <Название города | Название города | ...>" - добавляет один или несколько городов с указанным названием
    в список ранее добавленных городов. Города требуется вписывать через пробел!
2. "Информация" - выводит информацию обо всех городах в списке ранее добавленных.
3. "Информация_города <Название города>" - выводит информацию только для заданного города. Берёт
    уже существующую информацию из списка ранее добавленных городов.
4. "Информация_температура <температура>" - выводит информацию про города с указанной температурой.
5. "Удалить <Название города | Название города | ...>" - удаляет заданные города из списка ранее добавленных
    городов. Города требуется вписывать через пробел!
6. "Очистить" - очищает список ранее добавленных городов.
7. "Закончить" - завершает программу.
8. "Команды" - выводит список доступных команд.
